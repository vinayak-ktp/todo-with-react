# TODO List 🗒️  

A simple and functional TODO list application built with **React** and **Tailwind CSS**.  

## 🌟 Features  
- Add, update, and delete tasks easily.  
- Handle invalid or null inputs.
- Clean and simple UI for better usability. 

## 🛠️ Tech Stack  
- **React.js**  
- **Tailwind CSS**
